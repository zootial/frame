menu	菜单表(菜单：显示执行)
code	菜单编码
name	菜单名称
parentCode	父级菜单编码
linkUri	链接URI
icon	图标


file	文件表(文件：显示下载)
code	文件编码
name	文件名称
accessUri	访问URI
suffix	后缀
byteSize	byte大小


对象类型[ENTITY,FUNCTION,ELEMENT,MENU,FILE]
ENTITY类型拥有的动作：查询、修改、新增、删除、导入、导出
FUNCTION类型拥有的动作：执行、导入、导出、上传、下载
ELEMENT类型拥有的动作：显示、执行
MENU类型拥有的动作：显示、执行
FILE类型拥有的动作：显示、下载
